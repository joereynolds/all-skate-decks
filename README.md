## Why?

Because I want to try and find my old boards and didn't see a website like this so...

## Development

Everything uses docker. Run `./run.sh` and populate an `.env` file with valid
DB credentials and you should be good to go.

### Technologies used

- Laravel
- Livewire
- Tailwind
- AlpineJs
